0. head 태그 안에는 해당 문서의 내용에 관한 추가적인 정보가 들어간다.
    예를 들어,
    1) 해당 문서가 어떤 외부 파일과 연결되어 있는지, (예를 들어 우리가 연결한 외부 CSS파일)
    2) 어떤 문자열 인코딩을 사용하고 있는지 (meta charset)
    3) 전체적으로 어떤 내용을 담고있는지 (title)
    등이 있다.

1. div는 박스이다.

2. 마진은 박스와 박스 사이의 간격이고, 패딩은 박스와 그 안에 있는 내용물 사이의 간격이다.

2. class와 id라는 것이 있고, 모든 태그의 속성(attribute)로 설정할 수 있다. 
이것들을 설정하는 주된 이유는 CSS 선택자(selector)로 활용하여 원하는 태그를 선택하여 효과를 주기 위함이다.
class와 id에 대해 다시 설명. class보다 id가 더 '적용 우선순위가 높다'

3. HTML 파일에 CSS를 적용시키는 방법은 3가지가 있다.
    1) 외부 CSS파일을 HTML파일과 연결시키는 방법(head 태그 안에서 link태그 사용)
    2) 해당 HTML 파일 안에서 head 안에 style태그를 만들고 그 안에 CSS를 작성
    3) 각각의 HTML 태그의 속성(attribute)로 작성 
        예) <div style="margin:1px"></div>

4. emmet을 사용하면 빠르고 간편한 html 작성이 가능하다. 
    1) div*5 를 치고 엔터를 쳐보자
    2) div*5>input*3 을 쳐 보고 어떤 결과가 나오는지 확인
    3) div*2>input*3+div*3>img*2 도 해보자

5. lorem100 이라고 입력하면 아무말 100단어를 입력할 수 있다. 
    빈공간을 텍스트로 빠르게 채우고 싶을 때 사용한다. 100은 다른 원하는 숫자로 변경이 가능하다.



<예제에 사용된 태그 설명>

- a태그는 anchor 태그의 줄임말로, href 라는 속성(attribute)를 사용하여 원하는 다른 웹페이지로 연결이 가능하게 해 준다.

- div는 그냥 박스이다.

- hr은 구분선을 그어준다

- br은 줄바꿈을 해 준다

- div와 p의 차이 - div는 division의 약자로 말 그대로 그냥 공간을 나누기 위한 박스이고, 아무런 '의미'를 가지지 않는다. 무조건 가로 너비 전체를 차지하는 'block' 이라는 display property를 갖는다. 이것에 대해서는 월요일에 다시 설명할 것(block과 inline block, 그리고 inline이 무엇인지. div와 span의 차이)
 반면 p는 paragraph를 줄인 것으로, 말 그대로 '문단'을 의미한다. 따라서 p 태그 안에는 주로 텍스트를 작성하는 것이 옳으며, 기본적으로 상단과 하단에 마진(박스와 박스 사이의 간격)을 포함하고 있어서 사용하는 것 만으로도 해당 태그의 상단과 하단에 약간의 여백이 생겨나게 된다.
